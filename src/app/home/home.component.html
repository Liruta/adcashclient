Test work for AdCash

Hello and welcome to the test work I made for Addcash, I decided to use a backend in PHP with the framework CodeIgniter which is to me a good framework to make quick prototypes. I also decided to use Angular 5 as a front end since it is a framework I am learning on the side and i really much appreciate.

This app consist of creating and looking for orders. In this application you can:

  • Create / Edit / Delete Orders
  • Look for orders by user or product name
  • Limit the orders select for the current day or the last 7 days (or get them all)

Thanks for taking the time to try it and you can proccess by clicking the link "Orders" on the navbar
